# Chapter 02 - Operators in JavaScript

## What are Operators?

Operators are symbols used to perform operations on variables and values.

---

## 1. Arithmetic Operators

+  Addition  
-  Subtraction  
*  Multiplication  
/  Division  
%  Modulus  

Example:

```js
let a = 10;
let b = 5;

console.log(a + b);
console.log(a - b);
console.log(a * b);
console.log(a / b);
console.log(a % b);

2. Comparison Operators

== Equal to
=== Strict equal
!= Not equal

Greater than
< Less than

3. Logical Operators

&& AND
|| OR
! NOT
